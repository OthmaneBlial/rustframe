# Port entry queue observations

Badge verification is taking longer than the security staff estimated. The lane split itself clears quickly once trucks are released into the yard.

## Seen on site

- Manual badge checks stack up between 06:20 and 07:00.
- Drivers receive route direction only after the queue has already formed.
- Overflow staff appear only after the first wave is already delayed.

## Open question

Would pre-verified lane assignments remove the first queue without adding another checkpoint downstream?
