# Berlin warehouse walkthrough

The exception desk is logically close to the pick lanes, but visually it is almost invisible from the main path. New operators keep returning rejected items to the standard station.

## Quick fixes

- Add high-contrast overhead signage.
- Paint the exception path instead of relying on floor stickers alone.
- Include exception routing in the first-hour walk-through.
