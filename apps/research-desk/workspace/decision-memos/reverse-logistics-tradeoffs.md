# Reverse logistics tradeoffs

The team has been applying the same inspection depth to every return category. That keeps the SOP clean, but it is costing margin on low-value items.

## Recommendation

- Route premium accessories to repair-first review.
- Route basic returns to a fast resale or scrap decision after a short condition check.
- Keep one shared dashboard, but split the actual handling path.

## Notes

The labor threshold should be revisited monthly because courier pricing has moved twice this quarter.
