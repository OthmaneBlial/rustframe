RustFrame / Capability Demo

Direct native IPC. Embedded UI. No localhost bridge.

This demo proves the core runtime contract: a lightweight custom protocol, a tiny promise bridge, sandboxed file access, and allowlisted shell execution.

Transport Native IPC
Assets Embedded
Origin app://localhost

Window

Native controls

Filesystem

Sandboxed reads

The runtime only exposes the demo's `frontend/` directory. This reads `bridge.js` through the native API and rejects anything outside the allowed root.

Shell

Allowlisted process execution

The app can execute one pre-registered command: `listFrontend`. It runs without a shell and returns structured stdout, stderr, and exit code.
